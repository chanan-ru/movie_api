URL endpoints

Request URL HTTP Method
Return a list of ALL movies /movies GET
Return data of a single movie by title /movies/:title GET
Return data about a genre by title /movies/genre/:genreName GET
Return data about a director /movies/director/:directorName GET
Add new users /users POST
Update user's info /users/:id PUT
Add a movie to favorite list /users/:id/:favoriteMovie PUT
remove a movie from favorite list /users/:id/:favoriteMovie DELETE
Remove existing user /users/:id DELETE
